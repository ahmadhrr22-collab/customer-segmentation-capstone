## Project Overview
Proyek ini bertujuan untuk membantu tim pemasaran perusahaan ritel online dalam melakukan segmentasi pelanggan secara otomatis. Menggunakan algoritma **K-Means Clustering** dan analisis **RFM (Recency, Frequency, Monetary)**, kami mengelompokkan pelanggan ke dalam segmen yang actionable untuk strategi pemasaran yang lebih personal.

## Key Features
- **Automated RFM Calculation:** Menghitung skor Recency, Frequency, dan Monetary secara otomatis dari data transaksi mentah.
- **K-Means Clustering:** Mengelompokkan pelanggan menjadi 4 segmen utama (VIP, At Risk, Potential, Lost).
- **Interactive Dashboard:** Visualisasi 3D sebaran pelanggan dan download daftar pelanggan per segmen.

## Dataset
Dataset yang digunakan berasal dari UCI Machine Learning Repository: Online Retail Dataset.
Untuk tujuan demo, kami menyertakan file sampel `data_untuk_demo.csv`.
